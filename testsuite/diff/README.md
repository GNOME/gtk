BSD diff implementation
=======================

This directory contains code modified for GTK, based on the BSD diff
implementation from OpenBSD
([diffreg.c (rev 1.95)](https://cvsweb.openbsd.org/cgi-bin/cvsweb/src/usr.bin/diff/diffreg.c?rev=1.95)).
See the source files for copyright and licensing information.

When proposing modifications for these files, please consider whether
they are also suitable for submission to OpenBSD.
